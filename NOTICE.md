# LLMEX 고지와 귀속

LLMEX 자체 소스는 `LICENSE`의 MIT 조건으로 제공한다. Python 의존 패키지는 각 패키지의 별도
조건을 따른다. `llmex release bundle`의 SBOM은 설치 환경을 기록하지만 법률 의견을 대신하지 않는다.

원 한국어 Wikipedia 데이터는 패키지에 포함하지 않는다. Wikipedia 본문은 일반적으로 CC BY-SA와
GFDL 조건의 적용을 받을 수 있고 문서별 추가 고지가 있을 수 있다. 재사용자는 제목, page/revision ID,
source/dump URL과 날짜를 유지하고 배포 형태에 맞는 귀속 의무를 독립적으로 검토해야 한다.

학습 데이터 조건이 생성 가중치에 미치는 법적 효과를 이 프로젝트는 단정하지 않는다. 법무 검토와
명시적 공개 결정 전에는 모델 가중치, 전체 corpus, 생성 표본의 공개 배포가 금지된다.

`0.ref/`는 읽기 전용 참고 사본이며 wheel에 포함되지 않고 production 코드가 import하지 않는다.
