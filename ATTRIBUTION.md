# 한국어 Wikipedia 귀속 기록

LLMEX는 Wikimedia Foundation이 제공하는 날짜 고정 한국어 Wikipedia dump를 입력으로 사용할 수 있다.
원 데이터는 이 소프트웨어 배포물에 포함하지 않는다.

처리 corpus의 각 문서는 문서 제목, page/revision ID, 문서 source URL, dump URL과 날짜,
`CC BY-SA / GFDL; 문서별 추가 고지를 확인할 것`이라는 표식을 보존해야 한다.

실제 재배포자는 Wikimedia 이용 조건과 각 문서 이력을 확인하고 적용 가능한 저작자 표시, 링크,
동일조건변경허락 또는 GFDL 의무를 충족해야 한다. 이 파일은 법률 의견이나 배포 승인이 아니다.
