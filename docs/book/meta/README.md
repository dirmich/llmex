# 교재 제작 메타데이터

이 디렉터리는 기술 본문과 별도로 독자 약속, 문체, 출처, 검증 가능한 주장을 관리한다. 출판 형식·가격·플랫폼 정책처럼 바뀔 수 있는 정보는 기술 원고에 고정하지 않고 출판 직전 공식 자료로 다시 확인한다.

- [brief.md](brief.md): 독자, 범위, 완료 후 능력
- [style-guide.md](style-guide.md): 한국어 용어·문체·코드 표기
- [sources.md](sources.md): 현재 권위와 역사 자료, 외부 출처 경계
- [claims.md](claims.md): 실행·수치·구현 상태 주장 검증 원장
- [ai-usage.md](ai-usage.md): AI 보조 사용과 사람 검토 원칙

원고를 고칠 때 구현 상태나 수치 주장이 달라지면 본문과 `claims.md`를 함께 갱신한다.
