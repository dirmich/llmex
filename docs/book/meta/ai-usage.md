# AI 보조 사용 원칙

- AI는 목차, 초안, 코드 설명, 누락 후보 탐색을 보조할 수 있다.
- 모든 symbol·명령·수치·경로는 현재 소스, CLI, 테스트 또는 artifact로 사람이 재검증한다.
- AI가 만든 실행 결과, reviewer 서명, 법무 승인이나 출처를 사실처럼 생성하지 않는다.
- 원문·코드·그림의 license와 attribution은 AI 답변이 아니라 권위 자료에서 확인한다.
- 자동 생성 문장은 기술 reviewer와 한국어 편집 검토를 거친다.
- 변경 이력에는 어떤 파일과 검증 명령이 바뀌었는지 기록한다.
